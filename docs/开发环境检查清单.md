## 快速验证命令

将以下命令保存为脚本快速验证环境:

```bash
#!/bin/bash
# verify_env.sh

echo "🔍 验证开发环境..."

# Flutter
echo -n "Flutter: "
if flutter --version > /dev/null 2>&1; then
  echo "✓"
else
  echo "✗ 未安装"
fi

# Supabase CLI
echo -n "Supabase CLI: "
if supabase --version > /dev/null 2>&1; then
  echo "✓"
else
  echo "✗ 未安装"
fi

# 环境变量
echo -n ".env文件: "
if [ -f ".env" ]; then
  echo "✓"
else
  echo "✗ 不存在"
fi

# 依赖安装
echo -n "Flutter依赖: "
if flutter pub deps > /dev/null 2>&1; then
  echo "✓"
else
  echo "✗ 需要运行 flutter pub get"
fi

echo "✨ 验证完成!"
```

## 常见问题快速检查

如果遇到问题，按顺序检查:

1. **Flutter命令不可用**
   ```bash
   echo $PATH | grep flutter  # 检查PATH
   ```

2. **Supabase连接失败**
   ```bash
   echo $SUPABASE_URL  # 检查环境变量
   ```

3. **依赖安装失败**
   ```bash
   flutter clean
   flutter pub get
   ```

4. **模拟器无法启动**
   ```bash
   flutter devices  # 列出可用设备
   ```
